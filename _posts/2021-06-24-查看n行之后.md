平时使用tail都是查看最后几行日志：

```bash
tail -n 2 a.txt
```

其实以上命令相当于：

```bash
tail -n -2 a.txt
```

但tail还有一个用法，就是查看n行之后

```bash
tail -n +2 a.txt
```

怎么理解呢？

> -n, output the last NUM lines, instead of the last 10; or use -n +NUM to output starting with line NUM

NUM表示末尾NUM行，而+NUM表示从第NUM行开始。两者含义是完全不一样的，虽然都共用了-n这个flag。
